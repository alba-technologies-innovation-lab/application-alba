Alba Escape

Politique officielle du jeu Alba Escape.

1. Présentation du jeu

Alba Escape est un jeu Android de divertissement. Le jeu peut proposer des scores, récompenses, publicités ou contenus interactifs.

2. Données collectées

  • Données techniques Android.
  • Scores ou progression.
  • Données publicitaires.
  • Données statistiques.
  • Erreurs techniques ou crashs.

3. Utilisation des données

Les données servent à :

  • Faire fonctionner le jeu.
  • Sauvegarder certaines progressions.
  • Afficher des récompenses.
  • Améliorer les performances.
  • Corriger les bugs.

4. Récompenses virtuelles

Les récompenses, points ou contenus virtuels n’ont aucune valeur monétaire réelle sauf mention contraire.

5. Publicités

Le jeu peut afficher des publicités via Google AdMob.

6. Protection des mineurs

Le jeu ne doit pas être utilisé de manière excessive. Les parents sont responsables de l’utilisation par les mineurs.
