Bienvenue

Applications Alba est l’espace officiel dédié aux applications mobiles Android développées et publiées sous le compte développeur Alba.

Les applications peuvent proposer différents services selon leur nature : divertissement, outils numériques, musique, jeux, boutique, notifications, ou fonctionnalités interactives.

Voir mes applications sur Google Play

À propos des applications

Android

Les applications sont conçues pour fonctionner sur les appareils Android compatibles.

Expérience utilisateur

L’objectif est de proposer des applications simples, lisibles, rapides et faciles à utiliser.

Sécurité

Les services utilisés peuvent inclure des outils Google ou Firebase pour assurer le fonctionnement, la sécurité et l’amélioration des applications.

Politique de confidentialité

Cette politique de confidentialité s’applique aux applications Android publiées par Alba, sauf si une application possède une politique spécifique indiquée directement dans sa fiche Google Play ou dans l’application.

Données collectées

Selon l’application utilisée, certaines données techniques peuvent être collectées, comme le type d’appareil, la version Android, les données de diagnostic, les statistiques d’utilisation, les identifiants publicitaires ou les informations nécessaires au bon fonctionnement de certaines fonctionnalités.

Utilisation des données

Les données peuvent être utilisées pour faire fonctionner les applications, améliorer l’expérience utilisateur, corriger les erreurs, mesurer les performances, sécuriser les services, afficher des publicités ou envoyer des notifications lorsque l’utilisateur les accepte.

Services tiers

Certaines applications peuvent utiliser des services tiers comme Google Play Services, Firebase, AdMob, Google Analytics for Firebase ou d’autres outils nécessaires au fonctionnement de l’application.

Publicités

Certaines applications peuvent afficher des publicités via Google AdMob. Ces publicités peuvent utiliser des données techniques ou publicitaires conformément aux règles de Google.

Notifications

Certaines applications peuvent proposer des notifications push. L’utilisateur peut les accepter, les refuser ou les désactiver depuis les paramètres de son appareil.

Sécurité

Des mesures raisonnables sont mises en place pour protéger les données et limiter les accès non autorisés.

Droits de l’utilisateur

L’utilisateur peut demander des informations concernant ses données, leur suppression ou toute question liée à la confidentialité en contactant le développeur.

Contact

Pour toute demande liée aux applications ou à la confidentialité : contact.alba.play@gmail.com
