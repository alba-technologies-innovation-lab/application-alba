Les informations à joindre à votre message

  • 1. L’application concernée : indiquez clairement Zoé, PulseLove, Boutique 7 A&B, Alba Play ou Alba Escape.
  • 2. Votre objectif : expliquez ce que vous souhaitiez faire au moment où la difficulté est apparue.
  • 3. Le résultat observé : décrivez le message affiché, l’écran bloqué ou le comportement inattendu.
  • 4. Le contexte utile : précisez votre modèle de téléphone, votre version Android et, si possible, joignez une capture sans information sensible.

Demandes liées au compte ou aux données

Pour une suppression de compte ou de données, indiquez l’adresse e-mail utilisée dans l’application concernée. Une vérification raisonnable peut être nécessaire afin d’éviter qu’une autre personne demande la suppression de votre compte.

Ce qu’il ne faut jamais transmettre

Ne communiquez jamais votre mot de passe, un code de connexion reçu par SMS, les informations complètes de votre carte bancaire ou tout document contenant plus de données personnelles que nécessaire.

Important : les délais de réponse peuvent varier selon la nature de la demande. Un message précis et complet permet généralement d’éviter plusieurs échanges supplémentaires.
